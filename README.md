# **Научиться учиться** 

------------------------ 

_ПРОЕКТНАЯ РАБОТА 2 для ЯндексПрактикум по специальности Веб-разработчик_ 

Сайт описывает главные проблемы в обучении, а также техники обучения для тех, кто любит прокрастинировать. 
Проект написан средствами html и css с соблюдением методологии web-разработки БЭМ для html, css, и организации файлов проекта.  
Содержит блоки: 
1. header.
2. description.
3. techniques.
4. video.
5. oakley.
6. feynman.
7. digits.
8. khan.
9. kaufman.
10. resourses.
11. footer. 
 
------------------------------------------------------------------------------------------------------------------------------------------------
